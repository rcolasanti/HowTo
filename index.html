How to

Git

Initlise git

git config --global user.email "developer.colasanti@gmail.com"

git config --global user.name "Ric"

Clone git

git clone https://github.com/rcolasanti/HowTo.git

or:

git remote add origin "server"

Add git

git add *

Commit git

git commit -m "Commit message"

Push git

git push origin master

Set up web page

From "Repository Settings",click on the Settings tab and scroll down to the GitHub Pages section.Then select the master branch source and click on the Save button.
